Frequently Asked Questions

How secure is Gecko OS?

Show Answer

Gecko OS implements a hardened architecture, including sandboxing, zero-trust execution layers, built-in rollback mechanisms, and optional anti-malware capabilities. But the OS is only as secure as the user is smart, so use common-sense.

What kind of applications can I install?

Show Answer

Gecko OS supports .deb packages, .apk Android apps, raw binaries, and native compilation of compatible scripts. GeckoW2N lets you convert web apps into native apps, giving you serious flexibility across platforms.

Can I dual-boot Gecko OS with Windows or Linux?

Show Answer

Yep. Gecko OS can be dual-booted along side other operating systems.

Show Answer

Is Gecko OS good for developers?

Show Answer

Absolutely. Out of the box, you get compilers, terminals, environments, and a very wide support for languages.

Can I customize the OS UI or behaviors?

Show Answer

Yes. The Gecko desktop environment is fully themable, and config files let you tweak almost everything. We support dynamic window tiling, custom hotkeys, and even programmable gestures on supported trackpads.

How often does Gecko OS get updated?

Show Answer

Gecko OS gets updated about a few times per month. However, some security updates can happen in as little time as an hour after notice.
